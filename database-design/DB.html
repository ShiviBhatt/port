Pet Inventory Management System

To create a Pet Inventory System which will help visualize all the data pertaining to a particular store such as the products available/pets available, sales, inventory and the expenditure towards each section. This system stores the details of the inventory, stock availability, updates the inventory based on the sales details, generates sales and inventory reports. It will help the shop owner to make decisions based on the historical data and manage the business efficiently.

Technologies Used: Microsoft SQL Server, Amazon RDS, E-R Modelling, PowerBI
